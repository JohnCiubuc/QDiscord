QDiscord

QDiscord is a Qt wrapper for the Discord API written in C++. It was written to easily allow the creation of Discord bots.
Currently, it is in an incomplete state and will be under constant development, so expect things to either not work or not be available.
Feel free to submit an issue on the GitHub for this project if you need specific functionality to be working before anything else is implemented.

Discord

Discord is a proprietary Voice over IP application designed for gaming communities and is developed by Hammer & Chisel.
